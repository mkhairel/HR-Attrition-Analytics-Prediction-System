# HR-Attrition-Analytics-Prediction-System

## Project Overview
This project aims to analyze employee attrition patterns using HR data and build a predictive model to identify employees at risk of leaving the company. The goal is to help HR departments improve retention strategies and reduce turnover rate.

## Objective
- Analyze employee attrition patterns
- Identify factors influencing employee resignation
- Perform HR business analysis using SQL
- Build interactive Power BI dashboards
- Develop predictive machine learning models
- Generate HR insights for retention strategies

## Dataset
- The dataset consists of 1,470 employee records with 35 HR-related attributes including age, department, job role, income level, overtime status, job satisfaction, and attrition status.

## Machine Learning Model 
- Random Forest Classifier
- Label Encoding
- Train-Test Split
- SMOTE for class balancing

## Model Performance
- Accuracy: ~86%
- Applied SMOTE to handle class imbalance
- Evaluated using confusion matrix and classification report

## Key Insight
- Sales department recorded the highest attrition rate
- Employees working overtime were more likely to resign
- Lower salary employees showed higher attrition tendency
- Job satisfaction influenced employee retention
- Younger employees demonstrated higher turnover rates

## Business Problem
- Employee attrition can result in increased recruitment costs, productivity loss, and operational disruption. This project aims to identify key factors contributing to employee turnover and develop a predictive solution that enables HR teams to proactively address retention risks.

## Project Impact
- The project provides HR decision-makers with a data-driven approach to monitor workforce turnover trends, identify high-risk employee groups, and support retention planning through interactive dashboards and predictive analytics.
